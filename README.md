# HAI 2026 Homecoming Draw

HAI 2026 홈커밍데이 경품 추첨용 로컬 웹앱입니다.

## 실행

1. `홈커밍 참여 인원.xlsx` 파일을 이 폴더에 둡니다.
2. `start-roulette.ps1`을 PowerShell에서 실행합니다.
3. 브라우저가 자동으로 열리면 화면 안내에 따라 TOP 10 후보 선정 후 룰렛을 진행합니다.

```powershell
.\start-roulette.ps1
```

## 명단 반영 규칙

- `홈커밍 참여 인원.xlsx`의 첫 번째 시트를 읽습니다.
- 학생 명단은 `B열` 이름, `C열` 참석 여부를 읽습니다.
- 졸업생 명단은 `H열` 이름, `I열` 참석 여부를 읽습니다.
- 참석 여부가 `O`, `Y`, `Yes`, `1`이면 포함합니다.
- 행사 중 엑셀을 수정한 뒤 화면의 `Refresh` 버튼을 누르면 다시 불러옵니다.
- 앞 게임 베네핏 이름은 최대 5명까지 입력할 수 있고, 해당 이름이 TOP 10 후보에 있으면 룰렛 면적이 일반 후보의 2배가 됩니다.
- 전체 당첨자는 5명이며, 같은 상품 기준으로 순차 추첨합니다.
